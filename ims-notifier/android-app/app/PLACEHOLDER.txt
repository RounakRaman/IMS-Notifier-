PLACE YOUR REAL google-services.json HERE
==========================================

This directory must contain a file named exactly:

    google-services.json

You download this file from the Firebase Console after creating your project
and registering the Android app with package name:

    com.rounak.imsnotifier

Steps:
  1. Go to https://console.firebase.google.com
  2. Create (or open) a project
  3. Click the Android icon to add an Android app
  4. Package name: com.rounak.imsnotifier
  5. Download google-services.json
  6. Copy it into this folder:  android-app/app/google-services.json

The Gradle build will fail until this file is present. This is by design,
so the app always talks to YOUR Firebase project, not a placeholder one.

(This PLACEHOLDER.txt file itself can be deleted.)
